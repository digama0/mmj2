      • checkUnifySearchExclude

        public boolean checkUnifySearchExclude(Assrt assrt)
        Search array of assertions to see if a given assertion should be excluded from the unification search process.

        Assuming that the number of exclusions is small, we're using an array. If the number were very large a hash table could be used, but the array is searched only during the first pass through the LogicalSystem Statement Table (see ProofUnifier.java).

        Parameters:
        assrt - the given assertion
        Returns:
        true if assertion should be excluded
