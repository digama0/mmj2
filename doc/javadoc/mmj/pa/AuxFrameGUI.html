Class AuxFrameGUI

  • java.lang.Object
    • mmj.pa.AuxFrameGUI
  • Direct Known Subclasses:
    HelpGeneralInfoGUI, RequestMessagesGUI, SearchOptionsHelp, SearchResultsHelp

    public class AuxFrameGUI
extends java.lang.Object
    A base class to display an auxiliary frame of information that is standalone from a main application, for example a help page or a quick-and-dirty query report.

    To use it, just build one with the default constructor, then override variables, such as frameText or frameTitle using the "setter" methods. Then buildFrame() and showFrame() -- see main(), as an example.
