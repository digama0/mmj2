Class Sym

  • java.lang.Object
    • mmj.lang.MObj
      • mmj.lang.Sym
  • Direct Known Subclasses:
    Cnst, Var

    public abstract class Sym
extends MObj
    Sym holds a declared Metamath symbol and is the base class for Cnst and Var.

    In mmj there is only one occurrence of each symbol, and this includes constants and variables. For example, there is only one "(".

      • isActive

        public abstract boolean isActive()
        Is Sym active?

        Cnsts are always active as they cannot be defined inside a scope level, but a Var defined in a scope level is "inactive" outside of that scope.

        The question "is Sym 'x' active" has relevance only in relation to a Stmt: only "active" Syms can be used in a given Stmt's Formula.

        Returns:
        is Sym "active"
