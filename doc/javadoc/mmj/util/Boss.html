Class Boss

  • java.lang.Object
    • mmj.util.Boss
  • Direct Known Subclasses:
    GMFFBoss, GrammarBoss, LogicalSystemBoss, OutputBoss, ProofAsstBoss, SvcBoss, TheoremLoaderBoss, TMFFBoss, VerifyProofBoss, WorkVarBoss

    public abstract class Boss
extends java.lang.Object
    Boss is the superclass of GrammarBoss, LogicalSystemBoss, etc, which are used by BatchFramework to "exercise" mmj2 in batch mode.

    Boss consists of the abstract "doRunParmCommand" and some common parameter validation functions used in sub-classes of Boss.
